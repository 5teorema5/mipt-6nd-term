## Репозиторий посвящён решению задач по вычислительной математике

### Требования

Для сборки и запуска проектов необходимо следующее ПО:

- ```CMake``` (версия 3.10 или выше)
- ```MinGW32``` (для сборки с MinGW) или ```Make``` (стандартный)
- ```Python 3.11```
- Компилятор с поддержкой ```C++11/14/17```

### Инструкция по сборке с помощью `MinGW32`

```bash
# Создание директории для сборки
mkdir build
cd build

# Генерация Makefiles с помощью MinGW
cmake -G "MinGW Makefiles" ..

# Компиляция проекта
mingw32-make

# Возврат в корневую директорию
cd ..

# Запуск Python скрипта для обработки результатов
python processing.py

# Очистка временных файлов сборки (опционально)
rm -rf build
```

### Инструкция по сборке с помощью стандартного `make`

```bash
mkdir build
cd build
cmake ..
make
cd ..
python processing.py
rm -rf build
```
