# Название проекта

Краткое описание проекта: что делает, для чего предназначен.

## Сборка и запуск проекта

### Требования
- CMake (версия 3.10 или выше)
- MinGW (с поддержкой C++11/14/17)
- Python 3.x
- Make

### Инструкция по сборке

```bash
# Создание директории для сборки
mkdir build
cd build

# Генерация Makefiles с помощью MinGW
cmake -G "MinGW Makefiles" ..

# Компиляция проекта
mingw32-make

# Возврат в корневую директорию
cd ..

# Запуск Python скрипта для обработки результатов
python processing.py

# Очистка временных файлов сборки (опционально)
rm -rf build
