# Лабораторная работа №3

Предлагается создать объект и решить задачу его нагревания/остывания.

 - Для начала нужно создать геометрическую модель *gmsh* и сохранить её в формате *.msh*/*.unv*.
 - Затем нужно конвертировать эту модель в формат OpenFOAM с помощью утилит:
```bash
// для .msh
gmshToFoam <имя_модели>.msh
// для .unv
ideasUnvToFoam <имя_модели>.msh
```
 - в `constant/polyMesh/boundary.c` нужно посмотреть имя поверхности, которое было определено при создании модели:
```bash
/*--------------------------------*- C++ -*----------------------------------*\
| =========                 |                                                 |
| \\      /  F ield         | OpenFOAM: The Open Source CFD Toolbox           |
|  \\    /   O peration     | Version:  2112                                  |
|   \\  /    A nd           | Website:  www.openfoam.com                      |
|    \\/     M anipulation  |                                                 |
\*---------------------------------------------------------------------------*/
FoamFile
{
    version     2.0;
    format      ascii;
    arch        "LSB;label=32;scalar=64";
    class       polyBoundaryMesh;
    location    "constant/polyMesh";
    object      boundary;
}
// * * * * * * * * * * * * * * * * * * * * * * * * * * * * * * * * * * * * * //

1
(
    hh
    {
        type            patch;
        physicalType    patch;
        nFaces          5152;
        startFace       83184;
    }
)

// ************************************************************************* //
```
В данном примере это `hh`.

 - Далее нужно изменить файл `0/T.c`, в котором нужно указать, какую начальную температуру имеет наша поверхность и до какой температуры она будет нагреваться:
```bash
/*--------------------------------*- C++ -*----------------------------------*\
| =========                 |                                                 |
| \\      /  F ield         | OpenFOAM: The Open Source CFD Toolbox           |
|  \\    /   O peration     | Version:  v1912                                 |
|   \\  /    A nd           | Website:  www.openfoam.com                      |
|    \\/     M anipulation  |                                                 |
\*---------------------------------------------------------------------------*/
FoamFile
{
    version     2.0;
    format      ascii;
    class       volScalarField;
    object      T;
}
// * * * * * * * * * * * * * * * * * * * * * * * * * * * * * * * * * * * * * //

dimensions      [0 0 0 1 0 0 0];

internalField   uniform 1000;

boundaryField
{
	hh
	{
		type		fixedValue;
		value		uniform 10;	
	}
}

// ************************************************************************* //
```
В данном примере начальная температура поверхности `hh` равна `10`, нагревается до температуры `1000`.

 - Запускаем решатель:
```bash
laplacianFoam
```
 - Визуализируем в *ParaView*:
```bash
paraview f.foam
```
 - Для наглядности можно добавить разрез модели, запускаем визуализацию и радуемся красивой картинке =)
