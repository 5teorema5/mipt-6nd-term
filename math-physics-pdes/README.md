# math-physics-pdes (PDEs - Partial Differential Wquations)

Данный репозиторий посвящён предмету "Численное решение задач математической физики", который является альтернативным курсом в 6 семестре ФАКТ.

## ПО

Рекомендую установить операционную систему **Linux** для удобной работы =) Я использую дистрибутив Ubunty.

## Установка OpenFOAM

Открываем терминал.
Автоматически добавит ключ подписи репозитория, информацию о репозитории для вашей версии Ubuntu и выполнит обновление списка пакетов (`apt-get update`).
```bash
wget -q -O - https://dl.openfoam.com/add-debian-repo.sh | sudo bash
```
Устанавливаем `openfoam2112-default`, это самый полный пакет, включающий всё необходимое: солверы, утилиты набор обучающих примеров (tutorials) .
```bash
sudo apt-get install openfoam2112-default
```
Для начала работы с *OpenFOAM* используем команду
```bash
openfoam2112
```
Чтобы прекратить сессионную оболочку, пользуемся командой
```bash
exit
```

## Установка ParaView

*ParaView* должен быть установлен вместе с *ParaFOAM*. Для проверки используйте команду
```bash
paraview --version
```
Если версия не отобразилась, используйте следующие команды для установки *paraview*
```bash
sudo apt update
sudo apt install paraview
```
